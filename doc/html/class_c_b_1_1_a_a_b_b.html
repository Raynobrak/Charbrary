Detailed Description

Represents an Axis-Aligned-Bounding-Box.

An AABB is a rectangle whose sides are aligned with the axis of the world in which it is defined. It's a rectangle that cannot rotate.

AABBs are commonly used to represent hitboxes. This class contains member functions that allow simple collision detection between AABBs.
