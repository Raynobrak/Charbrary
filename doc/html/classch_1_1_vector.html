Detailed Description

Represents a 2D vector.

Represents a 2D vector whose components are axis-aligned. This class can be used to represent physical properties such as

  • acceleration
  • velocity
  • position
  • size
  • force
  • etc...

Most operators are overloaded to simplify vector calculus.
